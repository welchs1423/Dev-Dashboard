# Dev Dashboard

주니어 개발자를 위한 기술 트렌드 및 맞춤형 채용 공고 큐레이션 대시보드입니다.

## 기술 스택

- Frontend: Next.js, Tailwind CSS
- Data Collection: Python (requests)

## 로컬 실행 방법

1. 필요 패키지 설치
   \`\`\`bash
   npm install
   \`\`\`
2. 데이터 수집 스크립트 실행 (공고 데이터 갱신)
   \`\`\`bash
   python scraper.py
   \`\`\`
3. 개발 서버 실행
   \`\`\`bash
   npm run dev
   \`\`\`
   접속 주소: http://localhost:3000
